# Squad Agents

This directory contains agent definitions translated from fabric patterns that are marked as **agent-preferred**.
Each agent includes:

- `agent.yaml` - minimal manifest for metadata and references
- `agent.md` - agent-mode wrapper instructions
- `system.md` - core task prompt (copied from the original pattern)
- `references/` - knowledge base documents used by the agent
